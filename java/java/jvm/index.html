1.1. Rocky Linux 9 / Almalinux 9 + Maven + OpenJDK25

		
dnf update -y
dnf install https://dl.rockylinux.org/pub/rocky/10/AppStream/x86_64/os/Packages/m/maven-openjdk25-3.9.9-3.el10_1.noarch.rpm

修改配置文件 /etc/java/maven.conf 

		
JAVA_HOME=/usr/lib/jvm/jre-25-openjdk
		
		
		
[root@production ~]# cat /etc/java/maven.conf 
JAVA_HOME=/usr/lib/jvm/jre-25-openjdk
