13.1. ImportError: No module named 'zlib'

问题：编译安装的python3.4.2使用中提示 ImportError: No module named 'zlib'

原因：这个原因是你没有安装zlib

解决方法：安装 zlib-devel 后重新编译安装

yum install zlib-devel

./configure --prefix=/srv/python-3.4.2
make -j8
make install
