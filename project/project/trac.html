过程 1.1. TRAC - source

  1. setup.py

    wget http://peak.telecommunity.com/dist/ez_setup.py
python ez_setup.py

  2. Trac

    wget http://download.edgewall.org/trac/Trac-1.1.1.tar.gz
tar zxvf Trac-1.1.1.tar.gz
cd Trac-1.1.1
sudo python ./setup.py install
cd ..
